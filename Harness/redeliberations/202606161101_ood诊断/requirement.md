## 诊断任务

对 OOD 文档进行多维度问题定位，涵盖以下几类问题：

### 1. 定义矛盾
OOD 文档内部存在的自相矛盾的定义或描述，包括前后表述不一致、相互冲突的约束等。

### 2. 事实错误
OOD 文档中存在的不符合技术事实的描述，如错误的版本号、不存在的功能、错误的技术断言等。

### 3. 逻辑错误
OOD 文档中存在的逻辑推理错误，包括因果关系倒置、不成立的推导、不符合常识的假设等。

### 4. 偏离需求文档
OOD 文档中的设计方案与 `01_requirement.md` 需求文档不一致、遗漏或违背需求描述的部分。

### 5. 偏离路线图 Phase 0 阶段
OOD 文档中与 `03_roadmap.md` 路线图的 Phase 0 阶段范围定义不一致、包含不应在本阶段实现的内容、或遗漏了本阶段应包含的内容。
