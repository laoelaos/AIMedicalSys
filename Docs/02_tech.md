## 2. 技术栈分层总览

```text
┌─────────────────────────────────────────────────────────────┐
│  终端层（前端）  Vue 3 + TypeScript + Vite + UI 组件库       │
├─────────────────────────────────────────────────────────────┤
│  网关与微服务层  Spring Cloud Gateway / 微服务架构（拓展）    │
├─────────────────────────────────────────────────────────────┤
│  业务服务层  Spring Boot 3 + Spring MVC + Spring Security   │
├─────────────────────────────────────────────────────────────┤
│  AI 能力层  Spring AI / LLM 部署调优 / 知识图谱 / RAG / MCP │
│              / Function Calling / AI Agent                  │
├─────────────────────────────────────────────────────────────┤
│  数据持久化层  Spring Data JPA / SQL-Toy / MySQL / PostgreSQL│
├─────────────────────────────────────────────────────────────┤
│  中间件与通信  REST API / JWT / WebSocket / SSE / MQTT       │
├─────────────────────────────────────────────────────────────┤
│  部署与运维  Nginx 静态部署 / Jar 独立运行 / 云原生部署      │
├─────────────────────────────────────────────────────────────┤
│  硬件与采集层（拓展）  STM32 / 传感器 / 医学影像设备         │
└─────────────────────────────────────────────────────────────┘
```
